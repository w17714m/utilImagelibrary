Util Library
=========

A small library that adds commas to numbers

## Installation

  `npm install @jwilli0508/wjcq_lib_util`

## Usage

    var numFormatter = require('@jwilli0508/wjcq_lib_util');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`
